CONVERSION FORMULAS:
─────────────────────────────────────────────────────────

HEX → DECIMAL (per byte):
  AA (hex) = 170 (decimal)

LITTLE-ENDIAN → BIG-ENDIAN:
  Reverse byte order of 8-byte chunks
  [A8 F4 97 9B 77 E3 F9 3F] → [3F F9 E3 77 9B 97 F4 A8]

BINARY → BASE64:
  Encode raw bytes using base64 algorithm

BINARY → HEX:
  Convert each byte to 2-character hex string

IEEE 754 INTERPRETATION:
  Bytes 0-7:   64-bit float (real part = 0.0)
  Bytes 8-15:  64-bit float (imag part = φ)
