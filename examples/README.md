# Binary Fusion Tap - Multi-Language Examples

This directory contains production-ready implementations of the Binary Fusion Tap algorithm in 7 programming languages, generated by the multi-language compiler.

## Generated Implementations

All implementations produce identical output for k=11:

```
K Parameter: 11
Seed Value: 1234567891011
Tap State: 0b10001111101110001111110110000100001000100011
ZPE Overflow: 0b111011
```

## Regenerating Examples

To regenerate all examples from the compiler:

```bash
cd ..
python3 language_compiler.py --all
```

To generate a specific language:

```bash
python3 language_compiler.py --language rust -o custom_name.rs
```

## Algorithm Details

Each implementation includes:

1. **Seed Generation** - Concatenate numbers 1 through k
2. **8-fold Heartbeat** - Bit-shift left by 3 (equivalent to × 8)
3. **Phase Offset** - Add k parameter
4. **ZPE Overflow** - XOR extraction for k ≥ 10
