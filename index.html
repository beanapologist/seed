Quick Start

Universal Deterministic Stream Generator (Python)

from gq import UniversalQKD

# Generate consistent pseudo-random byte streams
generator = UniversalQKD()
stream = next(generator)  # 16 bytes of deterministic data
print(stream.hex())  # '3c732e0d04dac163a5cc2b15c7caf42c'

Golden Ratio Coin Flip (Python)

from gq import GoldenRatioCoinFlip

# Leverage golden ratio for quasi-random sequences
coin = GoldenRatioCoinFlip()
for _ in range(10):
    print(coin.flip())  # Deterministic: 0, 1, 1, 0, 1, 0, 0, 1, ...

Test Vector Generation (Python)

from gq import GQS1

# Generate deterministic test vectors
vectors = GQS1.generate_test_vectors(10)
for i, vector in enumerate(vectors):
    print(f"Vector {i}: {vector}")

JavaScript/Node.js Example

const { UniversalQKD } = require('@beanapologist/golden-seed');

// Generate deterministic streams
const generator = new UniversalQKD();
const stream = generator.next().value;
console.log(stream.toString('hex'));
